# 프로젝트 히스토리: AI 밸런스 게임 🚀

## 📅 2026-02-02: 프로젝트 킥오프
- **파트너**: 고대권 (클라우드/데브옵스 전문가)
- **조력자**: 미키17 (실리콘밸리 시니어 개발자 AI)

### 💡 아이디어 빌드업 과정
1. **초기 제안**: 대권 님의 본업을 살린 클라우드 비용 모니터링, AI 에이전트 SRE 도구 등 제안.
2. **방향 전환**: "본업보다는 가볍고 대중적인 부수입"을 위해 웹/앱 기반 서비스로 전환 결정.
3. **타임킬링 전략**: 가계부처럼 복잡한 데이터 연동 대신, 사람들이 심심할 때 하기 좋은 '밸런스 게임' 선택.
4. **비용 최적화**: 이미지 생성이 아닌, AI 키워드 추출 + Unsplash API 검색 조합으로 API 비용을 0원에 가깝게 설계.

### 🛠 기술적 결정
- **Framework**: Next.js 15 (최신 성능 최적화)
- **AI**: Gemini-1.5-Flash (저렴하고 빠른 텍스트 생성)
- **DB/Auth**: Supabase (빠른 MVP 구축)
- **Deployment**: Vercel & LocalTunnel (테스트용)
